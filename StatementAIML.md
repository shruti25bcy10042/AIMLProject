Problem Statement:-
Library staff face problems like inefficient manual processes, difficulty tracking books, inaccurate records, and a poor user experience due to slow service. This leads to staff inefficiency, misplaced or lost materials, and challenges in integrating digital resources. 

Scope of Project:- 
Scope of this project is to automate the system of library working staff making it less time consuming and effective.

Target Users:- 
Libray Staff

Features:- 
It allows the library staff to:-
1. Add Book
2. Add Member
3. Borrow Book
4. Return Book
5. List All Books
6. List Available Books
7. List Borrowed Books
8. List All Members
9. Remove Book
10. Remove Member
